# Rong VideoPlayer 🎬

一款基于 **Electron**、**HTML5** 和 **FFmpeg** 打造的 macOS 原生质感桌面视频播放器。支持导入本地文件夹，以左侧目录树的形式快速浏览并播放包含 **MP4, MKV, RMVB, AVI, FLV, WMV** 等在内的所有主流视频格式，具有出色的视觉体验和极低的时延。

![Rong VideoPlayer 运行界面截图](./screen/iShot_2026-06-29_10.17.14.png)

> **亮点优势**：针对 Chromium 浏览器原生不支持的 `.mkv`、`.rmvb` 等格式，后端采用 `ffprobe` 自动探测编码并配合 `ffmpeg` 在本地进行**实时智能转码与封装**，既保证了播放的流畅度（秒开、支持拖拽进度条），又避免了对电脑 CPU 的持续高负载消耗。

---

## ✨ 核心特性

- **📂 媒体目录树**：点击“打开媒体文件夹”即可导入，左侧展示清晰的目录结构。自动过滤非视频文件，自动隐藏空目录。
- **🔍 智能搜索**：支持在左侧目录中实时搜索视频文件名，匹配到的项目将闪烁高亮，并自动展开父级目录。
- **⚡️ 强兼容性**：支持 MP4, MKV, RMVB, AVI, FLV, WMV, MOV 等几乎所有音视频格式。
- **⏳ 状态恢复**：软件重启时，自动恢复最后一次打开的媒体目录，自动记录最后播放的文件和精确到秒的播放进度。
- **🚀 快捷键支持**：
  - `空格键 (Space)`：播放 / 暂停
  - `左方向键 (←)` / `右方向键 (→)`：快退 15 秒 / 快进 15 秒
  - `上方向键 (↑)` / `下方向键 (↓)`：音量调大 5% / 音量调小 5%
  - `F`：切换全屏播放
  - `M`：切换静音模式
- **⏱️ 倍速播放**：支持从 0.5x 到 2.0x 的播放速度调节。
- **🕯️ 最近播放列表**：展示最近播放的 10 个视频文件及播放进度。点击可一键恢复播放，并支持一键“清空”播放记录。
- **🍏 macOS 原生适配**：隐藏系统默认的窗口栏，采用原生毛玻璃质感侧边栏、无边框沉浸式拖拽以及高颜值控制条设计。

## 🛠 架构设计

应用采用 **Electron 主进程（Node.js 后端） + 渲染进程（Web 前端）** 的双层架构：

1. **主进程 (Node.js)**：
   - 负责调用 macOS 的原生文件夹选择对话框。
   - 递归扫描目录树并建立层级结构。
   - 启动一个轻量级的本地 HTTP 服务器，拦截视频请求。
   - 使用本地 `ffprobe` 探测目标视频。如果视频可被 Chromium 原生播放，则使用 HTTP 206 范围请求进行流式读取；如果视频格式不兼容（如 rmvb、mkv 的部分编码），则实时生成 `ffmpeg` 转码指令，并输出为 Fragmented MP4 流式传输。
   - 保存播放进度和历史记录到本地文件系统。
2. **渲染进程 (HTML5 + CSS3 + JS)**：
   - 使用 CSS3 精心定制了一套 macOS 风格的暗黑玻璃拟态（Glassmorphic）界面，带有自动隐藏的悬浮控制板。
   - 提供完美兼容的自定义进度条。当播放转码流时，前端通过合并 `transcodeStartTime` 和 `<video>` 的播放时间，模拟出一致的播放进度，并提供流式定位。

## 🚀 快速开始

### 1. 环境依赖

确保您的 Mac 已安装：
- **Node.js** (推荐 v18+)
- **Homebrew** 并且安装了 `ffmpeg` (本程序内置检测系统 Homebrew 安装的 ffmpeg 路径为 `/opt/homebrew/bin/ffmpeg`)

若未安装 ffmpeg，可在终端执行：
```bash
brew install ffmpeg
```

### 2. 安装与运行

在项目根目录下依次执行：

```bash
# 1. 安装项目依赖（Electron 及 Electron-Builder）
npm install

# 2. 启动开发版播放器
npm start
```

## 📦 打包为 macOS 原生应用 (`.app` / `.dmg`)

如果您想直接将程序打包为可在 Mac 应用程序（Applications）中双击运行的独立原生客户端，可执行：

```bash
# 生成未签名的本地 macOS 原生 .app 包 (输出在 dist/mac/ 下)
npm run pack

# 编译为可分发的磁盘映像 .dmg 文件 (输出在 dist/ 下)
npm run dist
```
